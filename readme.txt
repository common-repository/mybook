Using WordPress as a Book-Writing Platform.

== Description ==

Edit your own book. Create chapters, sections, change the order, export you book, etc.

Next version:

* Book settings (margin, fonts, etc)
* Multiple books
* Multiple editors/authors
*	Export book formats

Visit: http://araujo.cc

"Simplicity is the ultimate sophistication" -- Da Vinci

== Installation ==

Upload the MyBook plugin to your blog, activate it, then click on the menu.
